# HM Korozyon - Kurumsal Web Sitesi & Yönetim Paneli Projesi

HM Korozyon firması için sıfırdan tasarlanıp geliştirilmiş; modern, dinamik ve tam yönetilebilir kurumsal web uygulaması portfolyo projesi. 

> **Not:** Bu depo, gerçekleştirilen projenin yeteneklerini ve kullanılan teknolojileri sergilemek amacıyla oluşturulmuştur. Özel ticari bir proje olduğu için kaynak kodları herkese açık olarak paylaşılmamıştır.

## 🚀 Proje Özeti ve Özellikler

Bu proje, bir firmanın kurumsal kimliğini en iyi şekilde yansıtacak ve içeriklerini kolayca yönetebilmesini sağlayacak şekilde geliştirilmiştir.

- **Modern ve Duyarlı (Responsive) Tasarım:** Masaüstü, tablet ve mobil cihazlarla tam uyumlu, kullanıcı dostu şık arayüz.
- **Karanlık/Aydınlık Tema:** Kullanıcının tercihine göre anında değişebilen, modern dinamik tema sistemi.
- **Çoklu Dil Desteği:** Türkçe ve İngilizce (TR/EN) dillerinde içerik sunma altyapısı.
- **Dinamik İçerik Yönetimi:** Sitedeki slider, ürünler, hizmetler ve diğer tüm kurumsal bilgilerin yönetilebildiği, kullanımı kolay Admin Paneli.
- **Güvenli Yönetici Erişimi:** Cookie tabanlı Authentication ile korunan güvenli yönetim arayüzü.
- **Gerçek Zamanlı İletişim (SignalR):** Altyapıda kullanılan SignalR teknolojisi ile anında veri akışı ve anlık bildirim kapasitesi.
- **SEO ve Performans:** Arama motoru optimizasyonu (SEO) standartlarına uygun etiketlemeler ve yüksek performanslı sayfa yükleme süreleri.

## 🛠️ Kullanılan Teknolojiler

Projenin geliştirilmesinde en güncel web teknolojileri kullanılmıştır:

- **Backend:** C# / ASP.NET Core (Razor Pages)
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Gerçek Zamanlı İletişim:** ASP.NET Core SignalR
- **Kimlik Doğrulama:** Cookie tabanlı Authentication
- **Tasarım Araçları:** Özel CSS (Tailwind/Bootstrap olmadan tamamen özelleştirilmiş stil yapısı) ve FontAwesome İkonları
